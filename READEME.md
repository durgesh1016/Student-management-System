# Student Management System

## Description
A simple Student Management System built with Python. This project allows users to manage student records using a menu-driven interface.

## Features
- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Count Total Students

## Technologies Used
- Python 3
- Dictionary
- Functions
- Loops

## How to Run

1. Clone the repository
2. Open the project folder
3. Run:

```bash
python student_management.py
```
